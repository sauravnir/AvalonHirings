# AvalonHirings

Avalon Hirings is a complete solution for providing housemaids services directly in just a click of a button.

## Features
- Role-based dashboards (Admin, Client, Employee)
- Secure registration with manual verification
- Worker hiring and job management system
- Performance rating and evaluation module
- Reports and complaints handling
- Digital payments via eSewa and Khalti
- Centralized admin control and monitoring


## Technologies Used

- Language/s : Python, Javascript , SQL
- Framework/s : Django , React.Js
- Database : MySQL
